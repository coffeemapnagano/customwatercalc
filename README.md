# コーヒー抽出水計算機 | Custom Water Calculator

コーヒーの味を左右する「水」の硬度や成分（マグネシウム・カルシウム・バッファ）を、狙い通りの数値に調整するための精密計算ツールです。

## 🚀 サイトURL
[https://coffeemapnagano.github.io/customwatercalc/](https://coffeemapnagano.github.io/customwatercalc/)

## ✨ 主な機能
- **水質成分計算**: マグネシウム、カルシウム、アルカリ度（重曹）の目標値から、必要な添加量を算出。
- **SCA基準対応**: 目標とする水質の指標を簡単に設定可能。
- **レシピ管理**: 自分好みの「抽出水レシピ」を作成・調整するためのシミュレーション。
- **レスポンシブデザイン**: スマートフォンやタブレットからも快適に操作可能。

## 🧪 使い方
1. 作りたい水の**目標総量**を設定します。
2. 目標とする **Magnesium(mg/L)**、**Calcium(mg/L)**、**Buffer(mg/L)** の数値を入力します。
3. 計算結果に基づき、各溶液を添加して理想のコーヒー専用水を作成します。

## ☕ 開発の背景
「コーヒーの成分の98%以上は水である」という事実に着目し、世界中のバリスタが実践している水質コントロールを、より手軽に計算できるように開発しました。特に、サードウェーブコーヒーにおける酸味と質感のバランスを最適化するためのツールを目指しています。
